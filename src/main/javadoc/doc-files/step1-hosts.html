Step 1 - Set hosts entry

Instructions to set /etc/hosts hostname on target machine.
  1. Determine IP address
    $/sbin/ifconfig

  2. Set hosts entry:
    1. Edit the hosts file
      $sudo vi /etc/hosts

    2. Add entry to file:
      192.168.1.83 fortressdemo2

    3. Save and exit

  3. Verify 
            $ ping fortressdemo2
        PING fortressdemo2 (192.168.1.83) 56(84) bytes of data.
        64 bytes from fortressdemo2 (192.168.1.83): icmp_req=1 ttl=64 time=0.106 ms
