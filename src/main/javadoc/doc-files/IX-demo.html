Section IX - Fortress Demo #2

This section describes how to install and use the Fortress Demo #2 Web application

  1. Download: Fortress Demo #2 Package

  2. Set JAVA_HOME: 
                    
    
export JAVA_HOME=/opt/jdk1.7.0_10/
  3. Extract: 
                    
    
$JAVA_HOME/bin/jar -xf fortressdemo2-master.zip
  4. change directory: 
                    
    
cd fortressdemo2-master/
  5. Set M2_HOME: 
                    
    
export M2_HOME=../builder/apache-maven/
  6. Set ANT_HOME: 
                    
    
export ANT_HOME=../builder/apache-ant/
  7. To deploy, run this command: 
                        
    
# $M2_HOME/bin/mvn tomcat:deploy
  8. To redeploy: 
                        
    
# $M2_HOME/bin/mvn tomcat:redeploy
  9. You may use:

    1. non-SSL: http://localhost:8080/fortressdemo2, http://fortressdemo2.com:8080/fortressdemo2

    2. SSL: https://localhost:8443/fortressdemo2, https://fortressdemo2.com:8443/fortressdemo2

  10. Use Apache Solenium to drive Firefox through a range of security unit tests. Firefox must be installed before running this step. 
                    
    
$M2_HOME/bin/mvn verify -DskipTests=false
  11. Try the various user accounts:

    1. superuser/password

    2. poweruser/password

    3. user1/password

    4. user1_123/password

    5. user1_456/password

    6. user1_789/password

    7. user2/password

    8. user2_123/password

    9. user2_456/password

    10. user2_789/password

    11. user3/password

    12. user3_123/password

    13. user3_456/password

    14. user3_789/password
