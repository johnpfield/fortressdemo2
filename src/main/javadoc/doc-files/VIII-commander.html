Section VIII - Commander RBAC Web Admin

This section describes how to install and use Commander Web application

  1. Download: Commander RC39 Package

  2. Set JAVA_HOME: 
                    
    
# export JAVA_HOME=/opt/jdk1.7.0_10/
  3. Extract: 
                    
    
# $JAVA_HOME/bin/jar -xf commander-1.0-RC39.jar
  4. change directory: 
                    
    
# cd commander-1.0-RC39/
  5. Set M2_HOME: 
                    
    
# export M2_HOME=../builder/apache-maven/
  6. Set ANT_HOME: 
                    
    
# export ANT_HOME=../builder/apache-ant/
  7. Modify fortress.properties 
                        
    
# vi src/main/resources/fortress.properties
  8. Change the host, port and ssl config to match settings from Step V, Fortress Install 
                        
    
# Host name and port of LDAP DIT:
host=fortressdemo2.com
port=636

# Used for SSL Connection to LDAP Server:
enable.ldap.ssl=true
enable.ldap.ssl.debug=true
# This must point to truststore from Step 3:
trust.store=/home/smckinn/fortress/builder/src/test/resources/certs/mytruststore
trust.store.password=changeit
  9. Save and exit.

  10. Modify pom.xml tomcat-maven-plugin configuration to: 
                        
    
http://localhost:8080/manager/text
    Note: this step is temporary fix until RC40 releasey.

  11. Save and exit.

  12. To deploy commander, run this command: 
                        
    
# $M2_HOME/bin/mvn tomcat:deploy
  13. To redeploy: 
                        
    
# $M2_HOME/bin/mvn tomcat:redeploy
  14. You may use:

    1. non-SSL: http://localhost:8080/commander, http://fortressdemo2.com:8080/commander

    2. SSL: https://localhost:8443/commander, https://fortressdemo2.com:8443/commander

  15. Use Apache Solenium to drive Firefox through a range of security unit tests. Firefox must be installed before running this step. 
                    
    
$M2_HOME/bin/mvn verify -DskipTests=false
  16. Try the various user accounts:

    1. test/test


    2. test1/test

    3. test2/test

    4. test3/test
