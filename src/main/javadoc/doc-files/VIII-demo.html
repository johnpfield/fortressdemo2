This application uses Apache Wicket Web app and Mybatis database access frameworks to demonstrate application security techniques.

  1. Download package: https://github.com/shawnmckinney/fortressdemo2/archive/master.zip
  2. Extract
  3. Set JAVA_HOME
  4. Set M2_HOME
  5. Set ANT_HOME
  6. Run this command from the root package: 
                        
    
$ mvn install
  7. Run this command from the root package: 
                        
    
$mvn tomcat:deploy
    9. Go to this URL: http://localhost:8080/fortressdemo2
