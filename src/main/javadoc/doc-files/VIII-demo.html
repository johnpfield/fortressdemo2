This application uses Apache Wicket Web app and Mybatis database access frameworks to demonstrate application security techniques.

  1. Download: Fortress Demo #2 Package
  2. Set JAVA_HOME: export JAVA_HOME=/opt/jdk1.7.0_10/
  3. Extract: $JAVA_HOME/bin/jar -xf fortressdemo2-master.zip
  4. change directory: cd fortressdemo2-master/
  5. Set M2_HOME: export M2_HOME=../builder/apache-maven/
  6. Set ANT_HOME: export ANT_HOME=../builder/apache-ant/
  7. Run this command from the root package: 
                        
    
$ mvn install
  8. Run this command from the root package: 
                        
    
$mvn tomcat:deploy
    10. Go to this URL: http://localhost:8080/fortressdemo2
