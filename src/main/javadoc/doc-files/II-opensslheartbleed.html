Section II - Heartbleed Bug



This section provides instructions to fix the OpenSSL Heartbleed bug. Follow these steps if OpenSSL installed to target machine is version 1.0.1 through 1.0.1f.

To determine if upgrade is needed, enter the following at the command prompt: 
    

openssl version -b
If the date is less than Mon Apr 7 2014, and the version equals 1.0.1, then the machine is running an OpenSSL version that is vulnerable to the Heart Bleed bug and must be upgraded.

Steps to upgrade OpenSSL

  1. Update your repo (Debian): 
                
    
sudo apt-get update
  2. From the system command prompt (Debian): 
                
    
sudo apt-get upgrade openssl
